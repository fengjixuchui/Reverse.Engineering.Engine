# Reverse.Engineering.Engine

Additional Material for the RevEngE (Reverse Engineering Engine) paper.

## Instalation

Source your GDB:

```C
source install.sh
```

Compile test files:

```C
chmod +x tests/compile_tests.sh
./tests/compile_tests.sh
```

## Basic Usage

Test functionalities:

```C
$ gdb <binary>
(gdb) revtest
```
